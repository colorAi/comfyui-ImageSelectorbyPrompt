\# ComfyUI-Qwen-Image-Selector 🎨



一个适用于Qwen Edit 2509的智能图像选择器节点，通过关键词选择备用图像为AI生成提供精准的图像参考。



\## ✨ 特性亮点



\### 🎯 智能关键词匹配

\- 自动解析提示词中的关键词，智能选择最相关的参考图像

\- 支持复杂提示词的多关键词识别与匹配



\### 🖼️ 批量处理能力

\- \*\*最多支持15张\*\*备用图像输入，满足复杂创作需求

\- \*\*单次生成1-5张输出\*\*，灵活控制生成数量

\- 高效批量处理，提升创作效率



\### 🔧 高度可定制

\- 自定义备用图像名称，便于管理和识别

\- 灵活的输出数量控制

\- 直观的参数配置界面



\### 🚀 创作加速

\- 为创作者提供快速参考内容指定

\- 减少手动选择时间，专注创意实现

\- 批量生成相同风格系列作品



\### 📖 使用方法

\- 以 Qwen Edit 2509  基础工作流为例

\- 连接最多15张备用参考图像

\- 输入提示词（提示词格式为 {@A}，{@B}，{@C}等，官方默认最多支持3张参考图，这里可以使用三个关键词，）

\- 设置输出数量（1-5张）将输出图像连接到 TextEncodeQwenimageEditPlus  节点即可

\- 自定义图像名称（可选）
